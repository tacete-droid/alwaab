نظام الزيارات الميدانية — Field Visit System

GPS + صور بعلامة مائية + توثيق كامل — Flutter Mobile + Laravel Backend

📱 تدفق الزيارة الميدانية — Visit Flow

1
بدء الزيارة — Start Visit
Flutter يلتقط GPS تلقائياً، يربط بالمشروع، يسجل وقت البدء وبيانات الموظف.
2
التصوير — Photo Capture
Camera API في Flutter تلتقط الصورة. يُضاف GPS الدقيق + Timestamp لـ EXIF metadata.
3
العلامة المائية — Watermark Overlay
على السيرفر (Laravel + Intervention Image): تُضاف العلامة المائية تشمل الاسم + التاريخ والوقت + إحداثيات GPS.
4
الرفع والتخزين — Upload to S3
الصورة الأصلية والمعلَّمة تُخزن في AWS S3. البيانات في PostgreSQL (visit_photos table).
5
إنهاء الزيارة + التقرير — Complete Visit
الموظف يضيف ملاحظات ويُنهي الزيارة. يُرسل إشعار للمدير فوراً عبر Reverb.
6
عرض على الخريطة — Map View
الإدارة ترى جميع الزيارات على Google Maps مع Markers لكل موقع وفلترة حسب الموظف أو المشروع.

الأمان والحماية — Security Best Practices

معايير أمان المؤسسات — Enterprise-grade security implementation

🔐
Authentication — المصادقة
Laravel Sanctum SPA tokens. Bcrypt password hashing. Rate limiting on /auth/login (5 attempts/minute). 2FA optional (TOTP).
🛡️
Authorization — التفويض
Spatie RBAC. Laravel Policies per model. Gate checks on every sensitive action. Middleware groups per role.
📋
Audit & Activity Logs
Spatie Activity Log tracks all model changes. Log includes: user, IP, action, before/after values, timestamp. Immutable logs stored in separate DB table.
🔑
Data Security — أمان البيانات
All sensitive fields encrypted at rest. S3 files served via signed URLs (1-hour expiry). Database backups encrypted (AES-256).
🌐
API Security
HTTPS only. CORS whitelist. API versioning (/v1/). Request signing for mobile. SQL injection prevention via Eloquent. XSS protection headers.
📱
Mobile Security — Flutter
flutter_secure_storage for tokens. Certificate pinning. No sensitive data in local storage. Biometric auth option. Root/jailbreak detection.
☁️
Infrastructure Security
Cloudflare WAF + DDoS protection. Private VPC for DB servers. Redis AUTH password. Environment secrets in .env (never committed). Docker secrets in production.
🇦🇪
UAE Compliance — الامتثال
Data residency in UAE region. TDRA compliance for data protection. Privacy policy in Arabic & English. GDPR-compatible data export/delete.
